Data

Sources

We looked at NBA statistics from 2013 to 2019. Our data from DougStats.com can be downloaded from the csv files in the "data" folder in our final.zip. The rest of the data we got from Basketball-Reference which can be grabbed by using the "basketball-reference_parser.ipynb" file in the "data" folder.

Column description from original data files

Each column header and description for 2013 - 2019 draft pick statistics

pk draft position of player (pick)
tm team player was drafted by
player player name
college school player played for prior to draft
yrs years active in NBA
G total games played since drafted
mp total minutes played since drafted
pts points scored since drafted
trb total rebounds since drafted
ast total assists since drafted
fg% amount of field goals scored / amount of field goals attempted
3p% amount of 3 point field goals scored / amount of 3 point field goals attempted
